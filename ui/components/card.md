# KCard

Container for grouped content: dog overview, training summary, health snapshots, services.

## Variants

- `default`
- `elevated`
- `outline`

## Conceptual Props

- `title?`: string
- `subtitle?`: string
- `icon?`
- `children`
- `onPress?` / `onClick?`

Uses spacing, radius, and shadow tokens from `design-tokens/design-system.json`.
